## Primeiros passos

Clone o projeto e acesse a pasta.

```bash
$ https://github.com/Luis-Felipe-N/planetas.git
$ cd planetas
```

Siga os passos abaixo:
```bash
# Instalando depedencias
$ yarn add ou npm install

# Rodando projeto
$ yarn dev ou npm run dev
```
